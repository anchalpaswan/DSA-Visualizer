Bubble Sort

Overview

Bubble Sort is a simple comparison-based sorting algorithm that repeatedly iterates through the array, compares adjacent elements, and swaps them if they are in the wrong order. It gets its name from the way smaller elements "bubble up" to the top of the array during each pass.

Algorithm

The Bubble Sort algorithm can be summarized in the following steps:

  • Start from the first element of the array.
  • Compare the current element with the next element.
  • If the current element is greater than the next element, swap them.
  • Continue this process for the entire array.
  • Repeat the above steps until the array is sorted.

Time Complexity

The time complexity of Bubble Sort is O(n^2), where n is the number of elements in the array. It is a comparison-based sorting algorithm that has a higher time complexity compared to more efficient sorting algorithms such as Merge Sort or Quick Sort. Bubble Sort is mainly used for educational purposes or for small arrays where its simplicity may be advantageous.

Applications

Bubble Sort has limited practical applications due to its inefficient time complexity. However, it can still be useful in certain scenarios, such as:

  • Teaching introductory programming concepts
  • Sorting small arrays where efficiency is not a priority
  • Implementing other sorting algorithms as a part of a more complex algorithm

Bubble Sort in C++


    #include <iostream>
    using namespace std;
    
    void bubbleSort(int arr[], int size) {
        for (int i = 0; i < size - 1; i++) {
            for (int j = 0; j < size - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    // Swap elements if they are in the wrong order
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }
    
    int main() {
        int arr[] = {5, 2, 9, 1, 5, 6};
        int size = sizeof(arr) / sizeof(arr[0]);
        bubbleSort(arr, size);
        cout << "Sorted array: ";
        for (int i = 0; i < size; i++) {
            cout << arr[i] << " ";
        }
        cout << endl;
        return 0;
    }
