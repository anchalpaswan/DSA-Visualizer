Quick Sort: A Divide-and-Conquer Sorting Algorithm

Overview

Quick Sort is a popular comparison-based sorting algorithm that follows the Divide-and-Conquer paradigm. It is widely used for its efficiency and is often faster than other popular sorting algorithms like Bubble Sort, Insertion Sort, and Selection Sort.

The basic idea behind Quick Sort is to choose a pivot element from the array to be sorted, partition the array into two sub-arrays based on the pivot, and recursively sort the sub-arrays. The key to the efficiency of Quick Sort lies in its partitioning process, which efficiently divides the array into smaller parts and reduces the number of comparisons needed to sort the elements.

Algorithm

The Quick Sort algorithm can be described using the following steps:

  1. Choose a pivot element from the array. The choice of pivot can greatly impact the efficiency of the algorithm. Common pivot selection techniques include selecting the first element, last element, or a random element as the pivot.
  2. Partition the array into two sub-arrays, such that all elements less than the pivot are on the left side and all elements greater than the pivot are on the right side. This is done by iterating through the array and swapping elements to their correct positions relative to the pivot.
  3. Recursively apply the Quick Sort algorithm to the left and right sub-arrays. This is done by calling the Quick Sort function recursively on the left and right sub-arrays until the entire array is sorted.

Quick Sort Implementation in C++


    #include 
    
    // Function to swap two elements
    void swap(int& a, int& b) {
        int temp = a;
        a = b;
        b = temp;
    }
    
    // Partition function for Quick Sort
    int partition(int arr[], int low, int high) {
        int pivot = arr[low]; // Choose the first element as pivot
        int i = low + 1;     // Index of smaller element
    
        for (int j = low + 1; j <= high; j++) {
            if (arr[j] < pivot) {
                swap(arr[i], arr[j]);
                i++;
            }
        }
    
        swap(arr[low], arr[i - 1]); // Swap the pivot to its correct position
        return i - 1;                // Return the index of the pivot
    }
    
    // Quick Sort function
    void quickSort(int arr[], int low, int high) {
        if (low < high) {
            int pivotIndex = partition(arr, low, high); // Get the pivot index
    
            // Recursively sort the left and right sub-arrays
            quickSort(arr, low, pivotIndex - 1);
            quickSort(arr, pivotIndex + 1, high);
        }
    }
    
    int main() {
        // Array to be sorted
        int arr[] = {5, 3, 8, 4, 2};
        int size = sizeof(arr) / sizeof(arr[0]);
    
        std::cout << "Original Array: ";
        for (int i = 0; i < size; i++) {
            std::cout << arr[i] << " ";
        }
        std::cout << std::endl;
    
        quickSort(arr, 0, size - 1); // Sort the array using Quick Sort
    
        std::cout << "Sorted Array: ";
        for (int i = 0; i < size; i++) {
            std::cout << arr[i] << " ";
        }
        std::cout << std::endl;
    
        return 0;
    }
