Depth-First Search

Overview

Depth-First Search (DFS) is a graph traversal algorithm that explores a graph or tree data structure by traversing as far as possible along each branch before backtracking. It uses a stack data structure to keep track of the vertices to visit.

Algorithm

The DFS algorithm can be implemented recursively or iteratively. Here is the iterative approach:

  • Choose a starting vertex and push it onto the stack.
  • While the stack is not empty:
    • Pop a vertex from the stack and visit it.
    • If the visited vertex has unvisited neighbors, push them onto the stack.
  • Repeat until the stack is empty.

Applications

DFS has many applications, including:

  • Finding connected components in a graph
  • Topological sorting of directed acyclic graphs (DAGs)
  • Solving mazes
  • Detecting cycles in a graph

Recursive DFS in C++


    #include <iostream>
    #include <vector>
    using namespace std;
    
    void dfs(vector<vector<int>> &graph, int start, vector<bool> &visited) {
        cout << start << endl;
        visited[start] = true;
        for (int i = 0; i < graph[start].size(); i++) {
            int vertex = graph[start][i];
            if (!visited[vertex]) {
                dfs(graph, vertex, visited);
            }
        }
    }
    
    int main() {
        // Example graph represented as an adjacency list
        vector<vector<int>> graph = {{1, 2}, {0, 3}, {0}, {1, 4}, {3}};
        vector<bool> visited(graph.size(), false);
        dfs(graph, 0, visited);
        return 0;
    }
