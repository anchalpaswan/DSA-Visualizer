Linked List

Overview

A linked list is a data structure that consists of a collection of nodes, where each node contains data and a reference (or pointer) to the next node in the sequence. Linked lists are used to store and manage collections of data elements that are not necessarily stored in contiguous memory locations. They allow dynamic allocation of memory and provide flexibility in inserting, deleting, and modifying elements compared to other data structures like arrays.

Types of Linked Lists

There are several types of linked lists, including:

  • Singly Linked List: Each node contains data and a pointer to the next node in the list.
  • Doubly Linked List: Each node contains data, a pointer to the next node, and a pointer to the previous node in the list, allowing traversal in both directions.
  • Circular Linked List: A variation of singly or doubly linked list where the last node's next pointer points back to the first node, forming a circular loop.

Operations

A linked list typically supports the following operations:

  • Insertion: Insert an element at the beginning, end, or a specific position in the list.
  • Deletion: Delete an element from the list based on its value or position.
  • Traversal: Traverse the list to access and process each node or element.
  • Search: Search for a specific element in the list.
  • Modification: Modify the value of a node in the list.
  • Size: Get the number of elements in the list.

Applications

Linked lists are widely used in various applications and scenarios, including:

  • Implementing other data structures like stacks and queues
  • Dynamic memory allocation and deallocation
  • Managing large data sets and databases
  • Implementing algorithms like LRU (Least Recently Used) cache replacement
  • Handling sparse data and representing graphs
  • Managing system resources and memory
  • Implementation of certain algorithms like merge sort and reverse algorithms

Linked List Implementation in C++


#include 

// Node class to represent a node in the linked list
class Node {
public:
    int data;
    Node* next;

    // Constructor
    Node(int value) : data(value), next(nullptr) {}
};

// Linked List class
class LinkedList {
public:
    // Constructor
    LinkedList() : head(nullptr) {}

    // Insert a new node at the beginning of the list
    void insertAtBeginning(int value) {
        Node* newNode = new Node(value);
        newNode->next = head;
        head = newNode;
    }

    // Delete a node with the given value
    void deleteNode(int value) {
        if (head == nullptr) {
            std::cout << "List is empty" << std::endl;
            return;
        }

        if (head->data == value) {
            Node* temp = head;
            head = head->next;
            delete temp;
            return;
        }

        Node* current = head;
        Node* prev = nullptr;

        while (current != nullptr && current->data != value) {
            prev = current;
            current = current->next;
        }

        if (current == nullptr) {
            std::cout << "Element not found" << std::endl;
            return;
        }

        prev->next = current->next;
        delete current;
    }

    // Display the linked list
    void display() {
        Node* current = head;
        while (current != nullptr) {
            std::cout << current->data << " -> ";
            current = current->next;
        }
        std::cout << "null" << std::endl;
    }

private:
    Node* head;
};

int main() {
    // Create a new linked list
    LinkedList list;

    // Insert nodes
    list.insertAtBeginning(3);
    list.insertAtBeginning(2);
    list.insertAtBeginning(1);

    // Delete node with value 2
    list.deleteNode(2);

    // Display the linked list
    list.display();

    return 0;
}
