Stack

Overview

A Stack is a linear data structure that follows the Last-In-First-Out (LIFO) principle, where the last element inserted into the stack is the first one to be removed. It can be imagined as a collection of elements arranged one on top of the other, and elements can only be added or removed from the top of the stack. Stacks can be implemented using arrays or linked lists, and they are widely used in computer programming and various applications.

Operations

A stack typically supports the following operations:

  • Push: Insert an element at the top of the stack.
  • Pop: Remove the element from the top of the stack.
  • Peek (or Top): Retrieve the element from the top of the stack without removing it.
  • IsEmpty: Check if the stack is empty or not.
  • IsFull: Check if the stack is full (applicable for stack implementations using arrays).

Applications

Stacks have a wide range of applications in computer programming and various domains, including:

  • Expression evaluation and syntax parsing
  • Function call and recursion management in programming languages
  • Undo/Redo functionality in applications
  • Backtracking algorithms
  • Memory management in computer systems
  • Browser history management
  • Implementing algorithms like Depth-First Search (DFS) and Backtracking

Stack Implementation in Python


class Stack:
    def __init__(self):
        self.stack = []

    def push(self, item):
        self.stack.append(item)

    def pop(self):
        if not self.is_empty():
            return self.stack.pop()
        else:
            return None

    def peek(self):
        if not self.is_empty():
            return self.stack[-1]
        else:
            return None

    def is_empty(self):
        return len(self.stack) == 0

    def is_full(self):
        # Stack implemented using a dynamic list in Python
        return False

    def size(self):
        return len(self.stack)

# Example usage
stack = Stack()
stack.push(1)
stack.push(2)
stack.push(3)
print("Stack:", stack.stack)  # Output: Stack: [1, 2, 3]
print("Peek:", stack.peek())  # Output: Peek: 3
stack.pop()
print("Stack after pop:", stack.stack)  # Output: Stack after pop: [1, 2]
print("Is empty?", stack.is_empty())  # Output: Is empty? False
print("Size:", stack.size())  # Output: Size: 2
