Selection Sort

Overview

Selection Sort is a simple comparison-based sorting algorithm that works by repeatedly selecting the smallest (or largest, depending on the implementation) element from the unsorted part of the array and swapping it with the element at the beginning of the unsorted part. It continues this process until the entire array is sorted. It is called "Selection Sort" because it involves selecting the smallest or largest element and placing it at its correct position in the sorted part of the array.

Algorithm

The Selection Sort algorithm can be summarized in the following steps:

  • Start with the first element of the array, which is considered the sorted part of the array.
  • Find the smallest (or largest, depending on the implementation) element in the unsorted part of the array.
  • Swap the smallest (or largest) element with the element at the beginning of the unsorted part of the array.
  • Move the boundary of the sorted part of the array one element to the right.
  • Repeat the above steps until the entire array is sorted.

Time Complexity

The time complexity of Selection Sort is O(n^2), where n is the number of elements in the array. It is a comparison-based sorting algorithm that has a higher time complexity compared to more efficient sorting algorithms such as Merge Sort or Quick Sort. However, it performs well for small arrays or in scenarios where the number of swaps needs to be minimized, as it always performs a fixed number of swaps for each element.

Applications

Selection Sort can be useful in certain scenarios, such as:

  • Sorting small arrays or lists
  • As a simple sorting algorithm for educational purposes
  • As a part of a more complex sorting algorithm or algorithmic approach
  • In situations where minimizing the number of swaps is more important than time complexity

Selection Sort in C++


    #include <iostream>
    using namespace std;
    
    void selectionSort(int arr[], int size) {
        for (int i = 0; i < size - 1; i++) {
            int min_idx = i;
            for (int j = i + 1; j < size; j++) {
                if (arr[j] < arr[min_idx]) {
                    min_idx = j;
                }
            }
            if (min_idx != i) {
                int temp = arr[i];
                arr[i] = arr[min_idx];
                arr[min_idx] = temp;
            }
        }
    }
    
    int main() {
        int arr[] = {5, 2, 9, 1, 5, 6};
        int size = sizeof(arr) / sizeof(arr[0]);
        selectionSort(arr, size);
        cout << "Sorted array: ";
        for (int i = 0; i < size; i++) {
            cout << arr[i] << " ";
        }
        cout << endl;
        return 0;
    }
